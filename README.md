## Modo de usar

- Baixe e instale o Python 3.9.5 no seu computador (https://www.python.org/downloads/)
- Abra o CMD
- Navegue até a pasta do projeto
- Rode o seguinte comando "python calculadora.py"
- O programa vai pedir 3 informações
    - Novo Anúncio
    - Pesquisar Anúncio
    - Sair
